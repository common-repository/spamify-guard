== Description ==

Spamify Guard self learning algorithm can detect evil spambot traffic on his own, and block it instantly.

PS: You'll need an [SpamifyGuard.com API key](https://app.spamifyguard.com/#/signup) to use it. We have a free plan; paid subscriptions are available for businesses and commercial sites.

== Installation ==

Upload the Spamify Guard plugin to your website, activate it, then enter your [SpamifyGuard.com API key](https://app.spamifyguard.com/#/signup).

1, 2, 3: You're done!
